# RentDirect Backend (starter)

## Run locally
1. Copy `.env.example` to `.env` and set your MySQL credentials (defaults use localhost).
2. Install dependencies:
   ```
   npm install
   ```
3. Initialize DB with `scripts/seed.sql` (use MySQL client).
4. Start server:
   ```
   npm run dev
   ```

API base: http://localhost:4000/api/v1
